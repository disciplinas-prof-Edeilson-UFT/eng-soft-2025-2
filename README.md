### PROJETO: TimerBook


#### DESCRIÇÃO DO PROJETO

O projeto TimerBook tem como objetivo ser um software para gerenciamento de suas leituras. Inclui Fetures que inclui cadastro de livros que o usuário esteja lendo, acesso a banco de dados de livros já cadastrados e sistemas de métricas e lembrentes para auxiliar o usuário na leitura de seus diversos livros e comensurar seu desempenho.

#### Projeto: 
Cinetick
O Cinetick é uma plataforma inovadora para reserva de ingressos de cinema, conectando salas de exibição a usuários de forma prática e intuitiva. Através do aplicativo, os cinemas podem cadastrar seus perfis com informações detalhadas, como endereço, filmes em cartaz, sinopses, trailers, horários das sessões, tipos de salas, preços e promoções disponíveis.
